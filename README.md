# Leeds Crime Analysis Project

## Overview

This project analyses crime data from Leeds using the UK Police API.
Python was used to collect, process and visualise crime data to identify patterns and trends.

## Tools Used

- Python
- Requests
- Pandas
- Matplotlib

## Data Collection

Crime data was collected from the UK Police API using Python requests.

## Analysis Performed

The project analysed:

- Crime categories
- Crime frequency
- Location patterns
- Trends within the dataset

## Key Findings

- Anti-social behaviour was the most frequently recorded crime category within the analysed Leeds dataset.
- The results highlight which crime categories appear most frequently within the selected area.
- Further analysis could investigate location and time patterns to identify potential crime hotspots and trends.

## Limitations

- The analysis uses reported crime data, which may not represent all incidents.
- The dataset represents the selected Leeds area and timeframe only.
- Further analysis could compare different locations and longer time periods.
